# basic2py ‚Äî Commodore BASIC to Python Transpiler

Best-effort transpiler from Commodore BASIC (2.0, 7.0, 65) to Python.

The companion to [py2basic](../transpiler.py) ‚Äî goes the other direction.

## Requirements

Python 3.8+, no external dependencies.

## Usage

```bash
python3 basic2py.py program.bas
python3 basic2py.py program.bas -o program.py
```

## What it handles

| BASIC construct | Python output |
|---|---|
| `FOR I = 1 TO 10` / `NEXT I` | `for i in range(1, 11):` |
| `WHILE` / `DO` / `LOOP WHILE` | `while cond:` |
| `IF NOT (cond) THEN GOTO n` | `if cond:` / `if/else` |
| `IF cond THEN BEGIN` / `BEND` | `if cond:` / `if/else` |
| `GOSUB n` / `RETURN` | `def func():` / `func()` |
| `PRINT a ; b` | `print(a, b)` |
| `INPUT var` | `var = input()` |
| `REM text` | `# text` |
| `END` | `sys.exit()` |
| `SLEEP n` | `time.sleep(n)` |
| `MOD(a, b)` | `a % b` |
| `STR$(x)` | `str(x)` |
| `CHR$(x)` | `chr(x)` |
| `ASC(x)` | `ord(x)` |
| `LEN(x)` | `len(x)` |
| `INT(x)` | `int(x)` |
| `ABS(x)` | `abs(x)` |
| `SQR(x)` | `math.sqrt(x)` |
| `SIN/COS/TAN/ATN/LOG/EXP` | `math.sin()` etc |
| `x ^ y` | `x ** y` |
| `x <> y` | `x != y` |
| `CHR$(34)` concatenation | Restored to `"` in strings |

Variable names are expanded:
- `A` ‚Üí `a`
- `A$` ‚Üí `a_str`
- `A0` ‚Üí `a0`
- `A0$` ‚Üí `a0_str`

Function names are recovered from `REM -- name` comments
(as generated by py2basic). Unnamed subroutines become `sub_NNN`.

## What it can't handle

- Arbitrary `GOTO` jumps that don't match known patterns
  ‚Üí preserved as `# TODO: GOTO n ‚Äî unresolved jump`
- `ON x GOTO n1, n2, n3` computed jumps
- `POKE`/`PEEK` (emitted as TODO)
- `SYS`, `LOAD`, `SAVE` (emitted as TODO)
- Graphics and sound commands

## Honest expectations

Programs written with **py2basic** round-trip very cleanly ‚Äî the
transpiler generates predictable patterns that basic2py recognises.

Real-world BASIC programs with creative use of GOTOs will produce
output with TODO comments marking the unresolved jumps. That's
expected and honest ‚Äî use the TODO markers as a guide for manual
cleanup.

## Example round-trip

```
Python ‚Üí py2basic ‚Üí BASIC 2.0 ‚Üí basic2py ‚Üí Python
```

FizzBuzz goes in, FizzBuzz comes out. Variable names change
(i‚Üía, m3‚Üíb, m5‚Üíc) but the logic is intact and the output
runs correctly.
